## **프로젝트 개요**  
Spotify 오디오 특성 데이터를 기반으로 사용자 맞춤형 음악을 추천하는 콘텐츠 기반 추천 시스템을 개발합니다.

### **개발 환경**  
- **언어 및 도구:**  
  - Python 3.9  
  - Jupyter Notebook / VSCode 
- **라이브러리:**  
  - NumPy, pandas, scikit-learn, TensorFlow, Spotipy, mlflow, streamlit, fastapi, apache-airflow 

### **요구 사항**  
- Spotify API 액세스 키  
- Python 필수 라이브러리 설치
